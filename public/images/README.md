# Images du projet

Veuillez placer les images suivantes dans ce dossier pour qu'elles soient affichées sur le site :

- `logo.png` : Le logo de Maison Vélocio
- `filipe.jpg` : La photo de Filipe (créateur)
- `velocio.jpg` : La photo historique de Paul de Vivie (Vélocio)

*Note : Vous pouvez utiliser les extensions .jpg, .jpeg, ou .png selon vos fichiers, mais assurez-vous que les noms correspondent à ceux utilisés dans le code (filipe.jpg, velocio.jpg, logo.png).*
